# Getting Started

## Welcome to Compute42

Compute42 is a modern, fast, and focused IDE tailored for Julia developers. This guide will help you get started with the essential features.

## First Steps

1. **Open a Project:** Use the File Explorer in the left panel to navigate to your Julia project folder
2. **Create New Files:** Right-click in the explorer to create new Julia files (.jl)
3. **Start Coding:** Click on any Julia file to open it in the editor
4. **Run Your Code:** Use the Run button in the editor toolbar to execute Julia files

## Key Features Overview

- **Code Editor:** Full-featured editor with Julia syntax highlighting
- **Language Server:** IntelliSense, code completion, and error checking
- **Integrated Terminal:** Julia code execution with real-time output
- **Package Management:** Visual package management interface
- **Debugging:** Built-in debugger with breakpoints and variable inspection
- **Project Management:** Automatic Julia project detection and environment activation
