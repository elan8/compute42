# Troubleshooting

## Common Issues

Here are solutions to common problems you might encounter while using Compute42.

## Language Server Issues

- **LSP Not Starting:** Check the Environment Info panel for LSP status
- **No Code Completion:** Ensure Julia and required packages are properly installed
- **Error Detection Not Working:** Restart the Language Server from the Environment Info panel
- **Slow Performance:** Check if Julia is running and LSP is active

## Package Management Issues

- **Package Installation Fails:** Check your internet connection and Julia registry
- **Package Not Found:** Ensure you're in the correct Julia environment
- **Version Conflicts:** Use the Package Manager to resolve dependency conflicts
- **Environment Issues:** Restart Compute42 to refresh the environment

## Performance Issues

- **Slow Startup:** Ensure Julia is properly installed and accessible
- **Memory Usage:** Close unused files and restart the application if needed
- **LSP Performance:** Check the Environment Info panel for LSP status
- **File Operations:** Large files may take longer to process

## Getting Help

- **Environment Info:** Check the Environment Info panel for system status
- **Log Files:** Check application logs for detailed error information
- **Restart Application:** Many issues can be resolved by restarting Compute42
- **Update Julia:** Ensure you're using a compatible Julia version

## System Requirements

- **Julia Version:** Julia 1.12.1 (bundled with Compute42)
- **Operating System:** Windows 10/11, macOS 10.15+, or Linux
- **Memory:** Minimum 4GB RAM recommended
- **Storage:** At least 2GB free space for Julia and packages
