# Terminal & REPL

## Julia Execution

Compute42 provides an integrated terminal for running Julia code with real-time output and interaction.

## Terminal Features

- **Integrated REPL:** Full Julia REPL integration with command history
- **Code Execution:** Run Julia files directly from the editor
- **Real-time Output:** See results immediately as code executes
- **Interactive Mode:** Interact with Julia variables and functions
- **Error Handling:** Clear error messages and stack traces

## Usage Tips

- Use the Run button in the editor to execute the current file
- The terminal shows all output, errors, and warnings
- Use Ctrl+C to interrupt long-running operations
- Terminal history is preserved between sessions
- Use the clear button to clear terminal output

## Terminal Shortcuts

- **Clear Terminal:** Clear button or Ctrl+L
- **Interrupt Execution:** Ctrl+C
- **Copy Output:** Select text and use Ctrl+C
- **Paste Commands:** Ctrl+V to paste into terminal

## REPL Features

- **Command History:** Use up/down arrows to navigate command history
- **Tab Completion:** Tab completion for Julia functions and variables
- **Multi-line Input:** Automatic multi-line input for complex expressions
- **Variable Inspection:** Access and modify variables from previous executions
